## fork-web

The `fork-web` directory contains the Web client-side application for the fork-recipes project.

### Getting Started
To get started with the `fork-web` client app, follow these steps:

1. Navigate to the `fork-web` directory:
  ```sh
  cd fork-web
  ```

2. Install the dependencies:
  ```sh
  pnpm install
  ```

3. Start the development server:
  ```sh
  pnpm run dev
  ```

4. Open your browser and go to `http://localhost:3000` to view the app.
